## Tips

| Tip | Details |
|-----|---------|
| **Any language works** | Cantonese, English, mixed — the bot adapts |
| **Any channel works** | Everything works in #一般. Other channels are shortcuts. |
| **Attach files** | PDFs, images, voice messages — just send them |
| **Be natural** | Talk like texting a friend. No commands needed. |
| **Morning auto-posts** | Weather at 8AM, news at 8:30AM — no action needed |
| **Rain alerts** | Hourly checks — alerts only when rain is coming |
| **It remembers** | Tell it facts and it remembers for next time |
| **Evening review** | 9PM review of tomorrow's schedule — automatic |
